1. Ejecuta el comando `npm run build` en el terminal.
2. Si el comando falla, revisar los mensajes de error proporcionados en la salida del terminal.
3. Indentifica y corrige los errores en el código fuente según los mensajes de error.


### Notificaciones adicionales:
- No modificar las reglas de slint.
